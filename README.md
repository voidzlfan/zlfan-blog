# zlfan-blog

## 从零开始搭建个人博客
  概述：一直想搭一个自己专属的个人网站，恰好阿里云和腾讯云搞秒杀活动，首单购买很划算，用298元买了3年最低配置的腾讯云服务器。
  
  为了方便搭建，我选择了windows， 配置为：Windows Server 2016 64位中文版 标准型S5 1核2GB 1Mbps
  
### 技术选型
* react@17
* redux
* react-router-dom
* antd@4
* less
* node
* express
* mongodb && mongoose
